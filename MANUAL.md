## 1. ダッシュボード

ログイン後に表示されるホーム画面です。

**表示される情報:**
- 投稿数（累計・今日）
- API使用状況（使用回数 / 上限）
- 最近の投稿一覧（直近5件）

**クイックアクション:**
- AI投稿を生成
- スケジュール管理
- ペルソナ設定
- 戦略管理

---

## 2. 投稿管理

**場所:** サイドメニュー「投稿」

### 投稿一覧

ステータスでフィルタリングできます：
- **下書き** - 作成済み・未公開
- **予約済み** - スケジュール設定済み
- **投稿済み** - X に公開済み
- **失敗** - 投稿エラー

### 新規投稿作成

3つのフォーマットから選択できます：

| フォーマット | 説明 | デフォルト文字数 |
|------------|------|--------------|
| **ツイート** | 通常の短文投稿 | 280文字 |
| **長文投稿** | 記事のような長文 | 5,000文字 |
| **スレッド** | 連続ツイート（2〜25件） | 各280文字 |

**操作:**
1. フォーマットを選択
2. 内容を入力（手動 or AI生成）
3. 「インプレッション予測」で効果を確認（任意）
4. 「下書き保存」または「今すぐ投稿」

## 3. AI投稿生成

投稿作成画面の「AI投稿生成」パネルから利用できます。

**設定項目:**

| 項目 | 説明 |
|-----|------|
| ジャンル | 投稿のテーマ（例: テクノロジー、ビジネス） |
| スタイル | カジュアル / プロフェッショナル / ユーモア / 情報提供 / モチベーション |
| 生成数 | 一度に生成する候補数（1〜10件） |
| カスタムプロンプト | 追加の指示（任意） |
| ペルソナ使用 | 有効にすると、設定済みペルソナの口調で生成 |
| スレッド長さ | スレッド形式の場合のツイート数 |

**使い方:**
1. ジャンルとスタイルを選択
2. 「AIで生成」をクリック
3. 複数の候補から好きなものを選択
4. 選択した内容がエディタに反映される

## 4. ペルソナ

**場所:** サイドメニュー「ペルソナ」

AI生成時の「キャラクター設定」です。ブランドの声を統一できます。

**設定できる項目:**

| 項目 | 例 |
|-----|-----|
| 名前 | テック太郎 |
| 説明 | AI・テクノロジーに詳しいエンジニア |
| 性格特性 | 分析的、情熱的、ウィットに富む |
| コミュニケーションスタイル | カジュアル / プロフェッショナル / アカデミック |
| トーン | フレンドリー / 権威的 / 会話的 |
| ターゲット | エンジニア、スタートアップ経営者 |
| 専門分野 | AI、Web開発、クラウド |
| 背景ストーリー | 10年以上のエンジニア経験... |
| 言語パターン | 「〜だと思う」「技術的に言えば」 |
| 例文 | 過去の投稿例を3件まで登録 |

**操作:**
- 複数のペルソナを作成可能
- 「有効化」で1つだけアクティブに設定
- AI生成時に「ペルソナを使用」をONにすると反映

## 5. コンテンツ戦略

**場所:** サイドメニュー「戦略」

投稿の方針・目標を設定し、AIに反映させます。

**設定できる項目:**

### 成長目標
| 項目 | 説明 |
|-----|------|
| 投稿頻度 | 1日あたりの目標投稿数 |
| インプレッション目標 | 目標表示回数 |
| フォロワー増加目標 | 期間内の目標フォロワー数 |
| エンゲージメント率目標 | 目標エンゲージメント率（%） |

### コンテンツ設定
| 項目 | 説明 |
|-----|------|
| コンテンツピラー | 主要テーマ（例: AI、スタートアップ、Web3） |
| コンテンツミックス | ツイート/スレッド/長文の比率（%） |
| 最適投稿時間 | 効果的な投稿時間帯 |
| ハッシュタググループ | テーマ別のハッシュタグセット |
| 避けるトピック | 投稿しないテーマ |
| 競合アカウント | 参考にするアカウント |

**AI推奨機能:** 「推奨を取得」ボタンで、AIが現在の戦略を分析し改善提案を返します。

## 6. テンプレート

**場所:** サイドメニュー「テンプレート」

よく使う投稿パターンを保存して再利用できます。

**作成方法:**
1. テンプレート名を入力（例: 朝の挨拶）
2. コンテンツパターンを入力
   - 変数は `{変数名}` で指定
   - 例: `おはようございます！今日の{topic}についてシェアします。{content}`
3. 使用する変数を登録（例: topic, content）
4. カテゴリで整理（任意）

**使い方:**
- スケジュール投稿で「テンプレート」を選択するとテンプレートから投稿を生成

## 7. スケジュール

**場所:** サイドメニュー「スケジュール」

投稿の自動配信を設定します。

### スケジュールタイプ

| タイプ | 説明 | 例 |
|-------|------|-----|
| **一回限り** | 指定日時に1回だけ投稿 | 2月25日 9:00 に投稿 |
| **繰り返し** | cron式で定期投稿 | 毎日9時と18時に投稿 |

### 投稿ソース

| ソース | 説明 |
|-------|------|
| AI生成 | プロンプトを元にAIが毎回新しい内容を生成 |
| オリジナル | 事前に書いた内容を使用 |
| テンプレート | 保存済みテンプレートから生成 |

**cron式の例:**
- `0 9 * * *` → 毎日9:00
- `0 9,18 * * *` → 毎日9:00と18:00
- `0 9 * * 1-5` → 平日のみ9:00

## 8. 分析・インサイト

**場所:** サイドメニュー「分析」（Basic以上）

### メインメトリクス
- 合計インプレッション数
- 合計いいね数
- 合計リツイート数
- 平均エンゲージメント率

### トレンドチャート
- 期間選択: 7日 / 14日 / 30日 / 90日
- インプレッション・いいね・リツイート・リプライの推移をグラフ表示

### 投稿タイプ別分析
- オリジナル / AI生成 / テンプレートの効果比較

### PDCA分析
- AIが自動で分析レポートを生成
- 改善提案を具体的に提示
- 「PDCA自動適用」を有効にすると設定を自動最適化

## 10. Auto-Pilot（全自動運用）

**場所:** 設定 → Auto-Pilot設定

全自動でX運用を行うモードです。

### 設定項目

| 項目 | 説明 | デフォルト |
|-----|------|---------|
| マスタースイッチ | Auto-Pilot全体のON/OFF | OFF |
| 自動投稿 | AIが自動で投稿を生成・公開 | ON |
| 1日の投稿数 | 1日に自動生成する投稿数 | 3件 |
| 画像自動生成 | 投稿にAI画像を自動添付 | ON |
| 自動フォロー | キーワードに基づき自動フォロー | OFF |
| フォロー対象キーワード | カンマ区切りで指定 | - |
| 1日のフォロー上限 | 1日にフォローする最大数 | 10件 |

### 自動投稿のタイミング
1日4回（8時、12時、17時、21時）に自動実行されます。

### 有効にする手順
1. 設定 → Auto-Pilot設定で各項目を設定
2. 「Auto-Pilot設定を保存」をクリック
3. ダッシュボードでAuto-Pilotを有効化

## 11. 設定

**場所:** サイドメニュー「設定」

### X API設定
X (Twitter) APIの認証情報を入力します。「接続テスト」で動作確認できます。

### AIプロバイダー設定
| 項目 | 説明 |
|-----|------|
| AIプロバイダー | Claude または OpenAI を選択 |
| Claude API Key | Anthropic のAPIキー |
| OpenAI API Key | OpenAI のAPIキー |

### 一般設定

**AI投稿の言語:** 日本語 / English / 中文 / 한국어 / Espanol

**AI投稿の最大文字数:**
- ツイート/スレッド: プリセット [140] [280] [500] [1000]
- 長文投稿: プリセット [1000] [3000] [5000] [10000]
- 数値の直接入力も可能

**その他:**
- デフォルトジャンル
- デフォルト投稿間隔
- 自動フォロー ON/OFF
- PDCA自動適用 ON/OFF

## よくある使い方

### 手動で投稿を作成・公開
1. 投稿 → 新規作成
2. フォーマット選択 → 内容入力
3. 「今すぐ投稿」

### AIで投稿を自動生成して公開
1. 投稿 → 新規作成
2. AI投稿生成パネルでジャンル・スタイルを設定
3. 「AIで生成」→ 候補を選択
4. 「今すぐ投稿」

### ブランドの声を統一する
1. ペルソナ → 新規作成 → 性格・トーンを設定
2. 「有効化」をクリック
3. AI生成時に「ペルソナを使用」をON

### 完全自動運用を始める
1. ペルソナを設定・有効化
2. 戦略を設定・有効化（コンテンツピラー、投稿頻度）
3. 設定 → Auto-Pilot設定 → 各項目を設定・保存
4. ダッシュボードでAuto-Pilot ON
